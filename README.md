# ASCII farm

A terminal based ascii-style simple farming simulator. Players
can farm and sell crops to the market and gain money.

## Features

**Plant** : Players can plant different seeds in the unoccupied field by selecting specifc seed and pressing `P`

**Harvest** : Players can also hervest fully-grown crops from the field.

**Farm** : An ascii style farm where charecters are crops!
Players can plant seeds and hervest crops once they have fully-grown.

**Shop** : Shop is used to buy seeds of different types of crop and sell harvested crops with in game command.

**Inventory** : A persistent inventory to store all the seeds and crops.

**Money** : Money is what that controls the game. Different crops have different prices as well as seeds.

## Controls

### Farm

| Key | Action |
|------|---------|
| W | Move Up |
| A | Move Left |
| S | Move Down |
| D | Move Right |
| 1 | Select Rice Seed |
| 2 | Select Wheat Seed |
| 3 | Select Corn Seed |
| P | plant Crop |
| H | Harvest Crop |
| M | Go To Market |
| Q | Save and Quit |

### Market

| Key | Action |
|------|---------|
| C | Open transaction code bar |
| F | Return to Farm |
